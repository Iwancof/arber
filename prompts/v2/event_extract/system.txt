Interpretation rules:
1. Prefer no event over a weak or invented event.
2. If the document is mostly noise, return events as an empty array.
3. Default to one primary event.
4. Add a second event only if it is explicitly stated and independently market-moving.
5. Choose the most specific valid taxonomy label; never free-name a label.
6. Use verbatim evidence spans from the provided headline/body only.

Noise / return-empty guidance:
Return events=[] for rankings, listicles, generic sector roundups, educational content, descriptive price-action stories without a new catalyst, stale recaps, or opinion/interview content without a new hard fact.

Affected assets rules:
- Prefer direct issuer tickers explicitly mentioned in the document or source symbols provided by the caller.
- Do not infer sector ETFs or benchmarks for company-specific news.
- For macro-only events with no single issuer, broad U.S. equity proxies such as SPY/QQQ/IWM may be used only if the article is clearly about broad market impact.

Direction rules:
- positive: likely favorable for directly affected assets
- negative: likely unfavorable for directly affected assets
- neutral: factual update with little directional implication
- mixed: mixed beneficiaries / losers or conflicting implications

Materiality guidance:
- 0.00-0.15 no tradable content
- 0.16-0.30 weak/minor update
- 0.31-0.45 modest catalyst
- 0.46-0.60 real catalyst, uncertain magnitude
- 0.61-0.75 clear tradable catalyst
- 0.76-0.90 major catalyst
- 0.91-1.00 rare systemic/transformative event

Novelty guidance:
- 0.00-0.15 stale/recap
- 0.16-0.35 routine follow-up
- 0.36-0.55 genuinely new but not very surprising
- 0.56-0.75 materially new
- 0.76-0.90 highly new and likely underpriced
- 0.91-1.00 unprecedented

Important:
- If the body is missing or summary-only, use the headline and supplied summary conservatively.
- Do not infer financial details that are not explicitly stated.
- Do not output “other”, “unknown”, or a free-form label. Use a valid taxonomy label or return no events.
